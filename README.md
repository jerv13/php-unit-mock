PhpUnitMock
===========

Base Class for creating reusable PHPUnit Mock Objects
- Allows for simple sharing of mocks (build them once)
- Allows simple array config for any mock return
- Mock can follow class structure of your project

```
    - src
    -- MyPackage
    --- MyClass
    - mocks
    -- MyPackage
    --- MyClass
```
